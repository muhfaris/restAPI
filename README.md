# restAPI
Sample restAPI with mongoDB

### Format

| Action | Method  | URI   |
|--------|---------|-------|
| CREATED | POST | /api/v1/articles |
| READ | GET | /api/v1/articles |
| UPDATE | PUT | /api/v1/articles/{ID} |
| DETAIL | GET | /api/v1/articles/{ID} |
| DELETE | DELET | /api/v1/articles/{ID} |
